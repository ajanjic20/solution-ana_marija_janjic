# Product Catalogue

Product catalogue made with React, TypeScript and Tailwind CSS.

- Shows products with pagination
- Search products by name
- Filter by category and price
- Product details page with images
- Keeps search and filters in the URL
- Login with DummyJSON
- Favorite products for logged-in users

## Run the project

npm install
npm run dev

## Tests

npm run test
